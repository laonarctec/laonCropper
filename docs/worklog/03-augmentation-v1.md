# 03. Synthetic Padding Augmentation (v1)

## 접근
기존 100장에 무작위 배경 여백 추가 → bbox를 축소 → 500장으로 증강.
bbox 평균 면적: 86.3% → **34.9%**.

## 배경 구성
- 30% 유채색, 30% 회색/아이보리, 40% 자연색 (나무/베이지)
- 노이즈 + Gaussian blur

## 학습 결과
- mAP@0.5: **0.9819**, Early stop at epoch 72 (best epoch 52)
- P=0.977, R=0.976

## 실샘플 테스트 — 개선되었으나 과적합 우려
- 6/6 탐지 성공
- 4/6 영수증만 정확히 크롭 ✅
- 사용자 평가: "영수증을 놓친 케이스 발생" + 일부 결과 사용 불가
- **과적합 진단**: Box Loss train↓ val→ (gap=0.3331, OVERFIT ALERT)

## 결론
100 고유 영수증 × 5 = 500이지만 콘텐츠 다양성이 없어 과적합 불가피.
가중치 삭제 후 원본으로 복원 → 다른 접근 탐색.
